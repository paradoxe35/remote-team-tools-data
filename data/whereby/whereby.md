## Overview

Whereby is the easiest video conferencing tool to use, designed for online meetings, collaborations, and screen sharing without downloads or installations. Unlike traditional video conferencing tools, Whereby doesn't require software downloads or account setup for guests—participants simply click a link and join directly in their browser.

## Features

### Easy Access & Setup
- **No Downloads Required** - Guests join meetings directly in the browser
- **No Account Needed** - Participants don't need to sign up to join
- **Simple Meeting Links** - Share a link and start meeting instantly
- **Customizable Meeting Rooms** - Create branded, persistent meeting spaces
- **Browser-Based** - Works in any modern web browser

### Core Meeting Features
- **Screen Sharing** - Integrated screen sharing for presentations
- **Meeting Recording** - Record sessions for later review
- **Chat** - Text chat during video calls
- **Virtual Backgrounds** - Professional backgrounds for home offices
- **Reliable WebRTC Technology** - Secure, customizable video infrastructure

### User Experience
- **Intuitive Interface** - No learning curve for new users
- **No Clunky Downloads** - Eliminate installation friction
- **Quick Join** - From link to meeting in seconds
- **Professional Appearance** - Clean, modern interface
- **Stable Connections** - Reliable call quality
