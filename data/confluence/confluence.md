## Overview

Confluence is a digital workspace by Atlassian that brings teams together to collaborate on documentation and knowledge management. It serves as a living documentation platform where remote teams can create, organize, edit, and maintain comprehensive knowledge bases and project documentation.

## Key Features

- **Collaborative Wiki**: Create structured wikis with hierarchical page organization and team contributions
- **Rich Text Editor**: Format documents with text, images, tables, code snippets, and embedded content
- **Real-time Collaboration**: Multiple team members can edit simultaneously with automatic conflict resolution
- **Templates**: Pre-built templates for meeting notes, project plans, decision logs, and more
- **Search & Discoverability**: Powerful full-text search across all pages and attached files
- **Permissions & Security**: Fine-grained access controls with page-level and space-level permissions
- **Version History**: Track changes with version history and the ability to restore previous versions
- **Integration with Jira**: Seamless integration to link documentation with development work and issues
- **Analytics**: Admin controls for monitoring usage, user engagement, and content insights
- **Data Security**: Rules-based data control to manage how users, apps, and external parties interact with content
- **Comment & Feedback**: In-document comments, mentions, and feedback threads for collaborative review

## Recent Updates (2026)

Atlassian has introduced new analytics settings allowing admins to control third-party analytics integration and enhanced data security policies for controlling content access and file downloads.

## Pricing

- **Free Plan**: Up to 10 users with core documentation features
- **Standard Plan**: $65/month for unlimited users with standard features
- **Premium Plan**: $130/month with advanced security and analytics
- **Enterprise Plan**: Custom pricing with dedicated support and compliance features
- **Data Center**: On-premise deployment for large organizations with custom licensing
