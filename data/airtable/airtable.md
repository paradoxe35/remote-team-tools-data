## Overview

Airtable is a low-code platform that bridges spreadsheets and databases, allowing teams to organize and manage data in powerful ways. It enables creation of custom applications with advanced automation without requiring programming knowledge.

## Key Features

- **Grid View**: Spreadsheet-like interface for data entry
- **Multiple Views**: Gallery, calendar, kanban, form, and timeline views of the same data
- **Fields & Relationships**: Create complex data relationships and linked records
- **Automations**: Build workflows with no-code automation builder
- **Scripting**: JavaScript support for custom logic
- **Extensions**: Add custom functionality with Airtable extensions
- **API**: Programmatic access to your data
- **Templates**: Pre-built templates for common use cases
- **Collaboration**: Real-time editing and commenting
- **Permissions**: Granular access control at workspace, base, and view levels
- **Integrations**: Connect with Zapier, IFTTT, and other services

## Pricing

- **Free Plan**: Up to 5 bases with limited records and fields
- **Plus**: $20/month per user (billed annually), includes unlimited records and advanced features
- **Pro**: $45/month per user, adds advanced automation and API access
- **Enterprise**: Custom pricing for large organizations with advanced security and compliance
